# Проект: Автоматизация резервного копирования PostgreSQL с шифрованием

## Цель
Создать надёжную систему резервного копирования баз данных PostgreSQL с шифрованием (GPG) и хранением в MinIO (S3-совместимое облако). Восстановление данных для проверки целостности бэкапов.

## Технологии
- Docker, docker-compose
- PostgreSQL
- MinIO (S3)
- GPG (асимметричное шифрование)
- Bash, cron

## Состав проекта
- `docker-compose.yml` — оркестрация контейнеров
- `backup-scripts/` — скрипты и Dockerfile для контейнера `backuper`
  - `backup.sh` — создание дампа, шифрование, загрузка
  - `restore.sh` — восстановление из бэкапа
  - `entrypoint.sh` — настройка контейнера при старте
- `keys/` — публичный и приватный ключи GPG (приватный используется только для восстановления)
- `data/` — временные файлы и логи
- `.env` — переменные окружения (не включать в репозиторий)

## Запуск

1. Клонировать репозиторий.
2. Запустить скрипт deploy.sh
